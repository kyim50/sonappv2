🐛 TROUBLESHOOTING
═══════════════════════════════════════════════════════════════

Backend won't start?
  ► Check if port 3001 is in use
  ► Verify Riot API key in server/.env

Riot API verification failed?
  ► Key expired (regenerate daily)
  ► Check no extra spaces in .env

League not detected?
  ► Make sure League is fully loaded
  ► Wait 3-5 seconds after opening

Agora errors?
  ► Enable "App ID only" in Agora console
  ► Verify App ID matches .env
